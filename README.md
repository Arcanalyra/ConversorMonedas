# 💱 Conversor de Monedas en Java

Aplicación de consola desarrollada en Java 17 que permite realizar conversiones de divisas en tiempo real mediante el consumo de la API de ExchangeRate.
Implementando arquitectura limpia por capas (model, service, util), uso de HttpClient para peticiones HTTP modernas, parseo de respuestas JSON con la librería Gson y gestión de dependencias con Maven.
El sistema ofrece un menú interactivo continuo que permite convertir múltiples monedas (USD, MXN, EUR, TRY, GBP, RUB, EGP, CZK y cualquier otra soportada por la API).
Maneja excepciones de forma segura, utiliza variables de entorno para proteger la API Key y sigue buenas prácticas de desarrollo orientadas a portafolio profesional.
Destacando separación de responsabilidades, código reutilizable y estructura escalable, convirtiéndose en un proyecto ideal para demostrar habilidades en integración de APIs REST.
Manipulación de datos JSON, control de flujo interactivo en consola.
